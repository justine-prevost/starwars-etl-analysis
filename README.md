# Star Wars ETL Project

# Project overview
This project is an ETL pipeline using Star Wars data (SWAPI).
A part of the configuration is found in a YAML file.
It includes data extraction, transformation and analysis with pandas also SQL queries using SQLite3.

# Technologies
- YAML
- Python
- Json
- Logging
- Pandas
- SQLite3
- Matplotlib

# ETL Process
- Extract data from API / files
- Transform and clean data
- Load into SQLite database

# Analysis
- Groupby, aggregation, pivot_table, sort_values, merge and plot
- SQL analysis queries like percentage, rank, sum case when, etc

# How to run
1. Clone the repo
2. Install requirements
3. Run the main script
